## Project Overview

This is a Next.js-based personal website for chadjs-site. The repository is currently in its initial setup phase.

## Development Commands

This project uses Next.js. Once dependencies are installed, the following commands will be available:

- `npm run dev` - Start development server (typically at http://localhost:3000)
- `npm run build` - Create production build
- `npm run start` - Start production server
- `npm run lint` - Run ESLint

## Architecture

The project follows Next.js conventions:
- App Router or Pages Router (to be determined based on implementation)
- TypeScript configuration expected based on `.gitignore` patterns
- Vercel deployment support indicated by `.vercel` in `.gitignore`
